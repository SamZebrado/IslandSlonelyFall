# 本地指南游戏 (Local Guide Game)

一个基于本地存储的游戏化自助觉察工具，帮助用户通过自我共情、状态观测、微习惯养成和优先级决策来提升自我认知和生活质量。

## 运行方式

1. 直接在浏览器中打开 `index.html`
2. 或使用本地静态服务器：`npx serve .` 或 `python -m http.server`

## 功能范围

### 五大模块

1. **共情小屋** - 自我共情与情绪觉察流程
2. **状态观测台** - 记录每日能量、压力、清晰度等状态
3. **微习惯工坊** - 创建和管理微习惯任务
4. **优先级决策岛** - 通过五道门进行任务决策
5. **回顾花园** - 查看历史记录、导出/导入数据

## 数据保存

所有数据默认保存在浏览器 `localStorage` 中，不会上传到任何服务器。

## 导入导出

- **导出**: 在回顾花园页面点击"导出 JSON"按钮
- **导入**: 点击"导入 JSON"按钮，选择之前导出的备份文件
- **清空**: 点击"清空全部数据"，需要二次确认

## 心理安全边界

- 本工具仅供自助觉察使用
- 不是心理治疗、医疗诊断或危机干预工具
- 如有严重困扰，请寻求专业帮助

## 当前限制

- MVP 版本，纯前端实现
- 无云同步功能
- 无密码保护
- 无数据加密
