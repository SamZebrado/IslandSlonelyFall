## P1.3 微习惯工坊轻量游戏化

### 种子/点灯视觉反馈

新增"表达调音台"功能，替代原来的简单表达方式选择：

**支持的沟通对象：**
- 自己
- 伴侣/朋友
- 同事
- 导师/上级
- 家人
- 暂时不适合沟通的人

**支持的表达方式：**
- 只记录
- 发消息
- 当面说
- 先约时间
- 只设边界

**支持的语气强度：**
- 很轻
- 温和
- 清楚
- 坚定
- 边界明确

**特色功能：**
- 根据选择组合生成3-5条自然中文表达建议
- 当面说场景显示动作/眼神建议
- 发消息场景显示发送前检查
- 暂不适合沟通的人默认建议只记录、延后、设边界

### 原创占位BGM

生成了3段原创占位BGM：
- cloud_harbor_theme.wav - 首页/云港入口（明亮、轻盈）
- empathy_room_loop.wav - 共情小屋（安静、舒缓）
- review_garden_loop.wav - 回顾花园（温暖、放松）

使用 `scripts/generate_bgm.py` 生成。

### 云港/Cloud Harbor视觉主题

天空渐变背景、浮动云朵动画、卡片悬停效果、毛玻璃效果。

## P1.3 微习惯工坊轻量游戏化

### 种子/点灯视觉反馈

每个微习惯显示为种子或小灯：
- 未完成：🌱 柔和灰蓝色
- 已完成：🌿 发光或变成浅绿色/暖黄色
- 已跳过：🌰 保持柔和低饱和，不要灰掉得像失败

### 温和反馈文案

**完成反馈：**
- "这一步很小，但它已经算数。"
- "你刚刚给今天点了一盏小灯。"
- "不用很用力，能做一点就很好。"
- "这颗种子今天被照看了一下。"

**跳过反馈：**
- "这次先放过也可以，记录下来就够了。"
- "今天不做，不代表这件事失败了。"
- "先保留这颗种子，等合适的时候再照看。"

### 今日轻量统计

微习惯工坊顶部显示：
- 今日点亮小灯数量
- 今日待照看数量
- 今日先放过数量

**注意：不做连续打卡streak、不做失败率、不做排行榜**

### CSS动画

- gentleGlow: 完成时的柔和发光
- seedPulse: 种子变成小草时的脉动
- fadeIn: Toast渐入

## 核心交互流程

### 共情小屋流程
1. 描述发生的事情
2. 选择感受（多选）
3. 选择需要（多选）
4. 写出请求
5. 选择表达调音台设置（对象、方式、语气）
6. 生成表达建议

### 五道门决策流程
1. 这件事真的需要做吗？ → 决定继续或删除
2. 能不能删除？ → 进一步确认
3. 能不能简化/委托/推迟？ → 找到简化方案
4. 现在做是否比其他事情更重要？ → 决定优先级
5. 最小下一步是什么？ → 得出具体行动

## 当前 MVP 限制

- 仅支持单用户本地使用
- 无多语言支持
- 无数据备份到云端
- 无同步功能
- 无密码保护
- BGM为占位素材，非专业音乐

## 后续可迭代方向

1. **数据安全**: 添加密码锁定和加密存储
2. **数据可视化**: 添加状态趋势图、习惯完成率图表
3. **提醒系统**: 定时提醒记录状态或完成习惯
4. **个性化**: 支持自定义情绪词表和需求词表
5. **导出增强**: 支持导出为 PDF、图片等格式
6. **主题定制**: 支持浅色/深色主题切换
7. **BGM升级**: 替换为更专业的音乐素材
