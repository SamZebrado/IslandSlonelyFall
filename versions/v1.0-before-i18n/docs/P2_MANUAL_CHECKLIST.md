# local-guide-game — P2 结束前人工验证清单

> 建议耗时：10–15 分钟  
> 建议记录格式：PASS / FAIL / NA + 1 句备注

---

## A. 首页与导航

### A1. 首页地点卡片
- [ ] 首页能正常加载
- [ ] 五个地点卡片都可见：
  - 共情小屋
  - 状态观测台
  - 微习惯工坊
  - 优先级决策岛
  - 回顾花园
- [ ] 每张卡片的 badge / 描述 / "适合..." 标签可读
- [ ] 不同地点的色调区分清楚，但不影响文字可读性

### A2. 基本导航
- [ ] 点击每个地点都能进入对应模块
- [ ] 返回首页或切换模块时无明显错位/白屏
- [ ] 当前所在模块有明确的视觉标识

---

## B. 核心流程验证

### B1. 共情小屋
- [ ] 能正常填写一条共情记录
- [ ] 保存后记录成功写入
- [ ] 保存后出现柔和反馈，不显得突兀
- [ ] 出现"下一站"引导

### B2. 状态观测台
- [ ] 能正常记录一条状态
- [ ] 保存后数据可在回顾花园中看到
- [ ] 标签/状态信息显示正确

### B3. 微习惯工坊
- [ ] 能新增一个微习惯
- [ ] 能记录一条习惯日志
- [ ] 习惯与习惯日志关联正常
- [ ] 完成后反馈文案正常，不带过强打卡感

### B4. 优先级决策岛
- [ ] 第一扇门默认高亮
- [ ] 五道门能从头到尾走通
- [ ] 当前门 / 已完成门 / 未开始门状态清楚
- [ ] 完成后结果页正常显示
- [ ] 完成后"下一站"引导正常出现

### B5. 回顾花园
- [ ] 顶部 summary 正常显示
- [ ] 记录按时间分组正常（今天/昨天/近三天/更早）
- [ ] 类型标签颜色区分清楚
- [ ] 空花园状态文案正常
- [ ] hover 微动效存在但不干扰阅读

---

## C. localStorage 与恢复

### C1. 刷新恢复
- [ ] 创建记录后刷新页面，数据仍在

### C2. 关闭再打开
- [ ] 关闭标签页后重新打开应用，数据仍可恢复

### C3. 多标签页
- [ ] 两个标签页数据变化能被检测到

### C4. 导入导出
- [ ] 导出完整 JSON 成功
- [ ] 导入合法 JSON 成功
- [ ] 导入非法 JSON 时不崩溃、不污染数据

## D. 无障碍与动效

### D1. 键盘导航
- [ ] Tab 键遍历主要元素
- [ ] focus-visible 轮廓清楚
- [ ] Enter/Space 触发按钮

### D2. reduced motion
- [ ] 启用 reduced motion 后动画减弱

### D3. 可读性
- [ ] 文字与背景对比足够
- [ ] 不同状态不只依赖颜色区分
