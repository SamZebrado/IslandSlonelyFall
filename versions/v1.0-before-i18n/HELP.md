## 关于数据

### 为什么刷新页面后数据不见了？

这可能是因为：
1. **浏览器localStorage被禁用** - 某些浏览器或隐私模式下localStorage不可用
2. **浏览器缓存被清空** - 清空浏览器数据会删除localStorage
3. **浏览器数据限制** - 某些浏览器对localStorage有容量限制，超出后可能自动清理

**解决方法：**
- 检查浏览器设置，确保允许localStorage
- 使用"导出JSON"功能定期备份数据
- 尝试使用其他浏览器

### 为什么换浏览器后看不到之前的数据？

**这是正常现象。**

本地指南使用浏览器localStorage存储数据，每个浏览器的localStorage是独立的。换浏览器或换设备不会自动同步数据。

**解决方法：**
- 使用"导出JSON"功能备份数据
- 在新浏览器中使用"导入JSON"恢复数据

### 为什么多个标签页会提示数据更新？

这是应用的正常功能。

当你在多个浏览器标签页中打开本地指南时，如果在一个标签页修改了数据，其他标签页会检测到变化并提示"数据已被其他标签页修改"。

这是为了防止不同标签页的数据互相覆盖。

**解决方法：**
- 刷新页面以获取最新数据
- 或忽略提示继续使用当前页面

### 为什么动画变少了？

这可能是因为你的系统设置了"减少动画"偏好。

本地指南支持 `prefers-reduced-motion` 媒体查询，当系统设置为减少动画效果时，应用会自动减少或禁用动画。

**解决方法：**
- 在macOS上：系统设置 → 辅助功能 → 显示 → 勾选"减弱动态效果"
- 或在浏览器中正常浏览，动画会自动恢复

### 为什么没有登录/同步功能？

本地指南是一个**纯本地应用**，设计初衷是：
- 无需注册账号
- 数据完全保存在本地
- 无需网络连接
- 保护用户隐私

如果你需要在多个设备间同步数据，可以：
1. 使用"导出JSON"功能备份
2. 使用iCloud Drive、Google Drive等云盘在不同设备间传递备份文件
3. 导入到另一个设备

## 关于使用

### 如何备份数据？

1. 进入"回顾花园"
2. 点击"导出JSON"按钮
3. 保存下载的JSON文件到安全位置

### 如何恢复数据？

1. 进入"回顾花园"
2. 点击"导入JSON"按钮
3. 选择之前备份的JSON文件
4. 数据将恢复到浏览器中

**注意：导入会覆盖当前数据，请谨慎操作。**

### localStorage满了怎么办？

localStorage通常有5-10MB的容量限制。

**解决方法：**
1. 进入"回顾花园"，检查是否有大量记录
2. 如果记录很多，可以"导出JSON"备份后"清空全部数据"
3. 然后重新开始，或导入部分数据

### 数据损坏了怎么办？

如果数据出现异常：
1. 应用会自动尝试从备份恢复
2. 如果自动恢复失败，可以手动"清空全部数据"后重新开始
3. 如果有备份文件，可以尝试"导入JSON"恢复

### 为什么应用加载很慢？

可能原因：
1. 首次加载时BGM音频需要下载
2. 浏览器缓存未启用
3. 设备性能较低

**解决方法：**
- 等待音频加载完成
- 关闭/重新打开浏览器
- 尝试使用其他浏览器

## 关于功能

### 五道门是什么意思？

五道门是优先级决策岛使用的决策框架：

1. **放下门** - 这件事真的需要做吗？
2. **简化门** - 能缩小它的规模吗？
3. **求助门** - 必须我亲自做吗？
4. **稍后门** - 现在必须做吗？
5. **专注门** - 这是最重要的事吗？

每道门都是一个反思问题，帮助你找到最合适的行动。

### 共情小屋是什么？

共情小屋基于"非暴力沟通"(NVC)框架，帮助你：
1. 客观描述让你困扰的情境
2. 识别和表达你的感受
3. 探索这些感受背后的需求
4. 获得温和的自我对话建议

它不是一个分析工具，而是一个觉察工具。

### 微习惯工坊的理念是什么？

基于《原子习惯》的理念：
- 把目标缩小到"不可能失败"的程度
- 专注于"今天能做的一小步"
- 通过视觉反馈（如点灯）获得激励
- 坚持每天做，而不是一次性完成很多

## 关于隐私

### 我的数据会被上传吗？

**不会。**

本地指南是纯本地应用，所有数据只保存在你的浏览器localStorage中。应用不会：
- 上传任何数据到服务器
- 使用任何分析或追踪服务
- 收集任何个人信息

### 这个工具是专业的心理咨询吗？

**不是。**

本地指南是一个自助觉察工具，不是：
- 心理咨询
- 心理治疗
- 医疗诊断
- 危机干预

如果你正在经历严重的情绪困扰、焦虑、抑郁等问题，请寻求专业帮助。
